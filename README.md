Requisitos de Software:

XAMPP: Este software es esencial para iniciar el servidor Apache, dado que la aplicación depende de su funcionamiento.
Proceso de Inicio de la Aplicación:

Acceso al Proyecto: Para iniciar una aplicación desarrollada en Symfony, primero debes acceder al directorio que contiene el proyecto correspondiente. Puedes hacerlo mediante la navegación en tu sistema de archivos.

Terminal de Comandos: Una vez dentro del directorio del proyecto, abre una terminal de comandos en tu sistema. Asegúrate de que estás en la ubicación correcta del proyecto.

Iniciar el Servidor Symfony: Para poner en marcha la aplicación, utiliza el siguiente comando: symfony server:start.
